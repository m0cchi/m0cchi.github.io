cmdparser

cmdparserとは，Rubyスクリプトにコマンドの実装を支援するライブラリです．

install 

                        gem install cmdparser

usage

Rubyスクリプトのコマンドとして，cmd，other，-o，helpを用意した． また，cmdコマンドが実行された場合のみ実行されるサブコマンドがsubcmdとなる．

filename:cmd.rb

                        reqiure 'cmdparser'

                        cmd = CmdParser.new do
                          on 'cmd' do |arg|
                            print "cmd->#{arg}\n"
                          end
                          on ['other','-o'] do
                            print 'other cmd\n'
                          end
                          on 'cmd','subcmd' do |x,y|
                            print "(#{x},#{y})\n"
                          end
                        end

                        cmd.on 'help' do
                          print "help!!!!\n"
                        end

                        cmd.invoke
